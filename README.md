# 실시간 주가 모니터 크롬 익스텐션 🚀

네이버 증권에서 실시간 주식 정보를 크롤링하여 브라우저 화면에 표시하는 고급 크롬 익스텐션입니다. 
카드 클릭으로 손쉬운 전환, 스마트한 자동 슬라이드, 완벽한 사용자 경험을 제공합니다.

## ✨ 주요 기능

### 📊 **실시간 주가 모니터링**
- 네이버 증권 Offscreen DOM 크롤링을 통한 정확한 실시간 주가 정보 수집
- Python 로직 기반 정밀한 변동률 계산 (현재가 ± 전일가)
- 상승(🔴)/하락(🔵) 색상 구분으로 직관적 시각화
- Progress Circle 로딩 표시 및 에러 복구 시스템

### 🎬 **스마트 슬라이드 시스템**
- **자동/수동 모드 토글**: 설정에서 동작 모드 선택 가능
- **자동 모드**: 2개 이상 종목 시 자동 카드 전환 (1-30초 설정 가능)
- **수동 모드**: 좌우 네비게이션 버튼 (‹ ›)으로 완전 수동 조작
- **페이지 인디케이터**: 현재 위치 표시 (1/3, 2/3 등)
- **네이버 로고 버튼**: ![N] 클릭으로 해당 종목 네이버 페이지 바로 이동
- **실시간 간격 조정**: 설정 변경 시 즉시 반영

### ⚙️ **고급 설정 관리**
- **다중 종목 관리**: 무제한 종목 추가(➕)/삭제(🗑️)/활성화 토글
- **드래그 앤 드롭 순서 변경**: ⋮⋮ 핸들로 종목 순서 자유 변경
- **최근 조회 기능**: 추가한 종목 자동 저장 및 빠른 재추가 (최대 10개)
- **표시 위치 설정**: 4방향 위치 선택 + 드래그 앤 드롭
- **업데이트 주기**: 1-60분 사이 자유 설정
- **표시/숨김 토글**: 카드 표시 상태 제어 및 저장
- **동작 모드 설정**: 자동/수동 모드 토글 및 관련 설정 활성화/비활성화

### 🔧 **기술적 특장점**
- **Offscreen 크롤링**: 사용자에게 노출되지 않는 백그라운드 DOM 크롤링
- **Chrome Alarms**: 안정적인 주기적 데이터 업데이트
- **실시간 동기화**: popup 설정 변경 시 모든 탭에 즉시 반영
- **드래그 앤 드롭**: HTML5 Drag API 기반 직관적 순서 변경
- **모드별 UI**: 자동/수동 모드에 따른 동적 인터페이스 변경
- **스마트 이벤트**: 중복 방지 및 이벤트 분리로 안정적 조작
- **타이밍 최적화**: 애니메이션과 타이머 동기화로 부드러운 전환
- **로딩 상태 표시**: Progress circle과 함께하는 부드러운 UX
- **에러 복구**: 타임아웃, 재시도 메커니즘으로 안정성 보장
- **데이터 일관성**: localStorage 기반 설정 영구 저장 및 실시간 동기화

## 설치 방법

1. 이 저장소를 클론하거나 다운로드합니다
2. Chrome 브라우저에서 `chrome://extensions/` 로 이동
3. 우상단의 "개발자 모드" 활성화
4. "압축해제된 확장 프로그램을 로드합니다" 클릭
5. 다운로드한 폴더를 선택

## 📖 사용 방법

### 🎯 **주식 종목 관리**
1. **종목 추가**
   - 익스텐션 아이콘 클릭 → 설정 창 열기
   - 주식 코드(6자리)와 종목명 입력 → ➕ 버튼 클릭
   - **최근 조회**: 이전에 추가한 종목들을 클릭으로 빠른 재추가

2. **종목 관리**
   - **활성화/비활성화**: 토글 버튼으로 표시 여부 제어
   - **종목 삭제**: 🗑️ 버튼으로 완전 삭제
   - **순서 변경**: ⋮⋮ 드래그 핸들로 종목 순서 자유 변경
   - **즉시 반영**: 설정 변경 시 브라우저에 실시간 적용

3. **최근 조회 관리**
   - **자동 저장**: 종목 추가 시 최근 조회에 자동 추가
   - **중복 제거**: 기존 항목은 맨 위로 이동 (최신순)
   - **빠른 삭제**: 항목 호버 시 × 버튼으로 개별 삭제
   - **스크롤 영역**: 2줄 초과 시 스크롤로 10개까지 관리

### 🖱️ **카드 인터랙션**
1. **모드별 동작** (2개 이상 종목 시)
   - **자동 모드**: 카드 클릭 기능 비활성화, 자동 슬라이드만 동작
   - **수동 모드**: 좌우 네비게이션 버튼 (‹ ›)으로만 조작 가능
   - **네이버 로고 버튼 ![N]**: 모든 모드에서 해당 종목 네이버 페이지로 이동

2. **수동 모드 전용**
   - **좌우 버튼**: 카드 양쪽 끝에 위치한 ‹ › 버튼으로 수동 전환
   - **정확한 조작**: 이전/다음 버튼으로 원하는 종목 정확 선택
   - **자동 슬라이드 비활성화**: 완전한 수동 제어

3. **스마트 보호 시스템**
   - **이벤트 분리**: 네이버 버튼과 모드별 기능 독립 동작
   - **중복 방지**: 빠른 연속 조작 시 안정적 처리
   - **1개 종목**: 모든 네비게이션 기능 자동 비활성화

### 🎬 **슬라이드 기능**
1. **동작 모드 선택**
   - **자동 모드**: 설정 시간에 따른 자동 슬라이드 (기본)
   - **수동 모드**: 좌우 버튼으로만 조작 가능
   - **모드 변경**: popup에서 토글 스위치로 선택

2. **자동 모드 (Auto Mode)**
   - 2개 이상 종목 시 자동 활성화
   - 간격: 1-30초 사이 자유 설정 (기본: 5초)
   - 실시간 간격 변경 가능
   - 중복 실행 방지로 정확한 타이밍 보장

3. **수동 모드 (Manual Mode)**
   - **좌우 네비게이션 버튼**: ‹ › 클릭으로 정확한 전환
   - **자동 슬라이드 설정 비활성화**: 수동 모드 선택 시 자동으로 비활성화
   - **완전 수동 제어**: 사용자가 원할 때만 전환

4. **공통 기능**
   - **페이지 인디케이터**: ●○○ 현재 위치 표시 (1/3, 2/3 등)
   - **네이버 로고 버튼**: ![N] 네이버 증권 페이지로 바로 이동
   - **순서 기반 표시**: popup에서 설정한 순서대로 카드 표시

### ⚙️ **고급 설정**
1. **표시 설정**
   - **위치 선택**: 4방향 위치 + 드래그 앤 드롭 이동
   - **업데이트 주기**: 1-60분 (기본: 2분)
   - **동작 모드**: 자동/수동 모드 토글 스위치
   - **자동 슬라이드**: 1-30초 (기본: 5초, 자동 모드에서만 활성화)

2. **종목 관리**
   - **드래그 앤 드롭**: ⋮⋮ 핸들로 종목 순서 자유 변경
   - **시각적 피드백**: 드래그 중 위치 표시 및 애니메이션
   - **즉시 반영**: 순서 변경 시 브라우저 카드에 실시간 적용
   - **영구 저장**: 브라우저 재시작 후에도 순서 유지

3. **표시 제어**
   - **표시/숨김**: 눈 모양 아이콘으로 토글
   - **상태 저장**: 새 브라우저 세션에서도 설정 유지
   - **최근 조회**: 자동 저장 및 빠른 재추가 시스템

## 🔧 문제 해결

### **❌ 주가 정보가 표시되지 않는 경우**
1. **네트워크 연결**: 인터넷 연결 상태 확인
2. **주식 코드 확인**: 정확한 6자리 숫자 입력 (예: 005930)
3. **종목 활성화**: popup에서 해당 종목이 활성화되어 있는지 확인
4. **개발자 도구**: F12 → Console에서 에러 메시지 확인
5. **수동 새로고침**: 🔄 버튼으로 강제 데이터 새로고침

### **🎬 슬라이드가 작동하지 않는 경우**
1. **동작 모드 확인**: popup에서 자동 모드로 설정되어 있는지 확인
2. **종목 수 확인**: 2개 이상 활성화된 종목이 있는지 확인
3. **슬라이드 간격**: 자동 모드에서 슬라이드 간격이 적절히 설정되었는지 확인 (1-30초)
4. **표시 상태**: 카드가 표시 상태인지 확인 (숨김 상태에서는 슬라이드 안됨)
5. **설정 저장**: 💾 버튼으로 동작 모드 설정이 저장되었는지 확인

### **🖱️ 수동 조작이 작동하지 않는 경우**
1. **동작 모드 확인**: popup에서 수동 모드로 설정되어 있는지 확인
2. **종목 수 확인**: 2개 이상 종목이 있어야 네비게이션 버튼 활성화
3. **좌우 버튼**: 카드 양 끝의 ‹ › 버튼을 클릭 (자동 모드에서는 숨겨짐)
4. **네이버 버튼**: ![N] 네이버 버튼은 모든 모드에서 정상 동작
5. **설정 저장**: 💾 버튼으로 수동 모드 설정이 저장되었는지 확인

### **⚙️ 익스텐션이 작동하지 않는 경우**
1. **Chrome 버전**: 최신 Chrome 브라우저 사용 확인
2. **개발자 모드**: `chrome://extensions/`에서 개발자 모드 활성화
3. **익스텐션 활성화**: 해당 익스텐션이 활성화되어 있는지 확인
4. **권한 허용**: 필요한 권한이 모두 허용되었는지 확인
5. **재설치**: 익스텐션 제거 후 다시 설치

### **🔄 설정이 저장되지 않는 경우**
1. **저장소 권한**: Chrome 저장소 권한 확인
2. **브라우저 용량**: 충분한 저장 공간 확인
3. **개인정보 모드**: 시크릿 모드가 아닌 일반 모드에서 사용
4. **수동 저장**: 💾 버튼으로 설정 수동 저장

### **⋮⋮ 드래그 앤 드롭이 작동하지 않는 경우**
1. **드래그 핸들 확인**: 각 종목 왼쪽의 ⋮⋮ 핸들을 클릭해서 드래그
2. **마우스 조작**: 핸들을 클릭한 상태로 다른 위치까지 드래그
3. **드롭 위치 표시**: 드래그 중 파란색 선으로 드롭 위치 확인
4. **순서 반영**: 드롭 후 브라우저 카드 순서가 변경되는지 확인
5. **페이지 새로고침**: 변경이 즉시 반영되지 않으면 F5로 새로고침

### **📋 최근 조회가 작동하지 않는 경우**
1. **종목 추가**: 주식 코드+종목명 입력 후 ➕ 버튼으로 추가해야 최근 조회에 저장
2. **중복 종목**: 이미 모니터링 중인 종목은 클릭해도 추가되지 않음 (정상 동작)
3. **삭제 버튼**: 항목에 마우스 올린 후 나타나는 × 버튼으로 개별 삭제
4. **스크롤 확인**: 10개 이상 시 최근 조회 영역에 스크롤바로 확인
